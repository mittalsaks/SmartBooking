## 🛠️ Tech Stack & Features
- **Backend:** ASP.NET Core 8 & PostgreSQL (Reliable Data Storage).
- **Frontend:** React 19 + TypeScript + Tailwind (Glassmorphism UI).
- **Media:** Cloudinary Integration (Permanent Image Hosting).
- **Notifications:** Gmail SMTP Integration (Automated Email Alerts).
- **Real-time UX:** Framer Motion for smooth animations and feedback.

## 🚀 Key Features
- **Smart Booking:** Handle service capacity and availability dynamically.
- **Admin Insights:** Comprehensive dashboard to track revenue and bookings.
- **Persistent Storage:** Images stay online forever using Cloudinary API.
- **Automated Communication:** Real-time customer notifications via email upon every booking.

## 🔐 Setup
1. **Clone:** `git clone https://github.com/mittalsaks/SmartBooking.git`
2. **Setup:** Add your `Cloudinary`, `JWT`, and `SMTP` keys in Render Environment Variables.
3. **Run:** `dotnet run` (Backend) & `npm run dev` (Frontend).
